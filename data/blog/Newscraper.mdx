## Installation Journey

Ready to embark on the Newscraper experience locally? Follow these simple steps:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/rajika97/newscraper-lk.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd newscraper-lk
   ```

3. **Backend Setup:**

   ```bash
   cd backend
   npm install
   npm run dev
   ```

4. **Your Newscraper backend** should now be running at http://localhost:3000.

5. **Frontend Setup:**

   ```bash
   cd ..
   cd frontend
   npm install
   npm run dev
   ```

6. **Your Newscraper frontend** should now be running at http://localhost:5173.
