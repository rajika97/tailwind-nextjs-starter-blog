## Installation

Excited to get Promptium running on your local machine? Follow these simple steps:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/rajika97/promptium-ai-prompt-sharing-app.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd promptium-ai-prompt-sharing-app
   ```

3. **Install Dependencies:**

   ```bash
   npm install
   ```

4. **Set Up Environment Variables:**
   Create a .env file in the root directory and configure your environment variables.

   ```bash
   GOOGLE_ID=[Your Google ID]
   GOOGLE_CLIENT_SECRET=[Your Google Client Secret]
   MONGODB_URI=[Your MongoDB URI]
   NEXTAUTH_URL=http://localhost:3000
   NEXTAUTH_URL_INTERNAL=http://localhost:3000
   NEXTAUTH_SECRET=[Your NEXTAUTH Secret]
   ```

5. **Run the Development Server:**

   ```bash
   npm run dev
   ```

6. **Explore Promptium:**
   Visit http://localhost:3000 to experience Promptium locally.
